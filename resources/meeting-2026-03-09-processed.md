## 2. 연구 방향

### 슬로건
**"AI가 현실에서 멈추지 않는 환경을 만드는 연구"**

시스템이 서비스의 의도(Intent)를 파악하고, 자원을 자율적으로 구성하여 AI 서비스 품질을 유지하는 기술

### 연구 목표
분산된 AIoT 디바이스들이 각자 가진 서비스 기능을 사용자의 의도(Intent)에 맞춰 자동으로 조합, 배치, 재구성하여 최적의 상태를 유지하는 지능형 시스템 연구

### 3대 연구 축: DATA / SYSTEM / SERVICE

#### DATA
- 다양한 환경에서의 데이터 수집 및 정제 프로세스 구축
- 키워드: #데이터수집 #빅데이터 #전처리

#### SYSTEM
- AI 기술을 핵심 동력으로 지능형 네트워크 및 시스템 구현
- 키워드: #AI시스템 #지능형네트워크 #EdgeAI

#### SERVICE
- 사용자 중심의 지능형 응용 서비스 개발
- 키워드: #지능형서비스 #IoT응용 #솔루션

### Problem → Challenge → Solution

1. **Problem: "AI도 환경이 중요합니다"**
   - 아무리 좋은 AI 모델도 그것을 받쳐주는 인프라와 네트워크 환경이 뒷받침되지 않으면 무용지물입니다.

2. **Challenge: "작지만 빨라야 합니다"**
   - 자원이 한정적인 소형 장치(Edge)에서도 상황을 즉각적으로 판단하고 반응하는 기술이 필수적입니다.

3. **Solution: "자율 지능 시스템"**
   - 복잡한 설정 없이도 시스템이 스스로 의도를 파악해 최적의 상태를 유지하는 기술을 연구합니다.

### 연구 로드맵 (학위별)

#### 학부생 → 경험
- **AI 기반 스마트 시티 클린 시스템 개발**
- 현업 기업과 협업하여 컴퓨터 비전 기반 지능형 폐기물 관리 시스템 개발
  - 비정상 사용행위 감지, 불법투기 감지, 종량제 봉투 용량 판단
- 핵심: 현실 환경에서 AI를 배포 및 운영하는 실전 경험

#### 석사과정 → 분석
- **Ad-hoc 네트워크 성능 예측 및 경로 복구 | IIoT 환경 이상 탐지 학습 프레임워크**
- 시계열 모델로 네트워크 성능 저하 패턴을 감지하고, 산업 환경의 희소 이상 데이터를 효과적으로 학습
- 핵심: AI 시스템 운영에 필요한 개별 핵심 기술 연구 (네트워크 예측, 이상탐지)

#### 박사과정 → 설계
- **Intent 기반 자율 서비스 기술 연구 (연구실 전체 방향 통합)**
- 5G/Wi-Fi 환경 AI 서비스 장애 조기 감지, 분산 엣지 멀티모델 추론, Closed-loop 자율 적응 제어
- 핵심: 모든 요소를 하나의 자율 시스템으로 통합 설계
